# Python Web App

## Overview
A simple Python web application served via the built-in HTTP server.

## Project Structure
- `py.py` - Main application entry point; runs an HTTP server on port 5000

## Running the App
The app runs automatically via the configured workflow. It listens on `0.0.0.0:5000`.

## Deployment
Configured for autoscale deployment using `python py.py`.
