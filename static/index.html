Slide 5
Model Antrean M/M/c — Landasan Teori
  • M/M/c: Markovian arrival, Markovian service, c servers (tungku)
  • λ = laju kedatangan nira (batch/menit) = 1 / mean_interarrival
  • μ = laju layanan tungku (batch/menit) = 1 / mean_service_time
  • ρ = utilisasi server = λ / (c·μ) → stabil jika ρ < 1
  • Syarat: ρ < 1 agar sistem tidak collapse (antrian tak terbatas)
  • Distribusi: Eksponensial (scipy.stats) — bukan pseudo-random bawaan Python
💡 Sertakan flow diagram: Nira Datang → Antrean → Tungku Masak → Output
Slide 6
Implementasi — Arsitektur Sistem
  • Frontend: HTML5 + Vanilla JS + Chart.js — form input, visualisasi grafik
  • Backend: Python FastAPI — REST API endpoint POST /api/simulate
  • Engine: SimPy (Discrete Event Simulation) + scipy.stats (distribusi eksponensial)
  • Alur: Form → JSON POST → FastAPI → SimPy Engine → JSON Response → Chart.js
  • Asynchronous: UI tidak freeze saat simulasi berjalan (loading spinner)
